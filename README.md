# PokeDex

PokeDex app in Vanilla JS using  for GUI and [pokeapi](https://pokeapi.co/api/v2/pokemon/) .

![](https://i.postimg.cc/fT5FNNks/Pokedex.jpg)
 
####  Features:
- Get Stats of pokemon
- Get type of pokemon
- Get humidity levels

#### External APIs Used:
- [PokeAPI](`https://pokeapi.co/api/v2/pokemon/)
